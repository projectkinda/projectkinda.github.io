Sound familiar?

Most productivity tools treat you like a to-do list. But the reason you're not getting things done isn't a missing feature — it's a pattern. There are six of them.

Activation failure

"I know exactly what I need to do. I just can't make myself start."

Decision paralysis

"I have so many things I could do that I end up doing none of them."

Perfectionism trap

"It's not ready yet. It needs to be right before I can send it."

Momentum dependency

"I work well — but only when I'm already in the zone. Getting there is the problem."

Burnout / depletion

"I used to be productive. Now even small tasks feel like too much."

Identity fog

"I'm busy all the time but I don't feel like I'm moving toward anything."

Kinda starts by figuring out which one is yours. Then everything — how it talks to you, how it breaks down your work, how it nudges you — adapts to that.
